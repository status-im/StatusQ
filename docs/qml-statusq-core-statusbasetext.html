    StatusBaseText QML Type

    Displays multiple lines of text. Inherits Text. More...

    Import Statement: import StatusQ.Core 0.1
    Since: StatusQ.Core 0.1
    • List of all members, including inherited members

    Detailed Description

    The StatusBaseText item displays text. For example:

    StatusBaseText {
    width: 240
    text: "Hello World!"
    font.pixelSize: 24
    color: Theme.pallete.directColor1
}

    For a list of components available see StatusQ.
